Creating a PFP

Before you can generate variations of your image, you need to generate a base image, which all generated images will be based off of. To start type in a prompt with keywords seperated by commas. Generate a few images using the prompt untill you find a decent image. Press save, to save the image and data. Then use the variation generator to fine-tune your image. Play around with the variation image seed to produce diffrent results.
